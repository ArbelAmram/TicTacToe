# TicTacToe
X/O Game (Three In A Line)

A Java game (Player vs CPU), where you try to completet three X's in a line,
and the CPU will try to complete three O's in a line.
You can try verticly, horizontly or diagnaly.
It's your desicion who will play the firt turn.

The Challenge:
I dare you to let the CPU play the firt turn, and still win, 
at best you will ecomplish a tie.
